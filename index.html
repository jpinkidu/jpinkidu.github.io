Mistwetter (weather app)

An Android app to display the current weather forecast. Including hourly forecast and a 7 day forecast.

Information the app displays:

  • temperatur (current, hourly, 7 day forecast)

  • location (at the moment hard coded position)

  • time (current)

  • humidity (current)

  • wind speed (current)

  • preciptation (current)

  • pressure (current)

  • summary (7 day forecast)

Forecast information are from forecast.io

Ubuntu 14.04 LTS: Install Android Studio

Get Android Studio from the official website.

When the download is finished, extract the file. Open the Terminal by using the shortcut Strg+Alt+T (Ctrl+Alt+T) and navigate to the directory, where you put the Android Studio files.

If you use a 64 Bit Ubuntu Version be aware that you have the following things installed:
sudo apt-get install lib32z1 lib32ncurses5 lib32bz2-1.0 lib32stdc++6

Go to /android-studio/bin/ and open studio.sh with ./studio.sh

When you choose Custom Installation you can change the Theme of Android Studio. Check the files you want to install and click on Next and at last on Finish. Everything will be installed immediately.
